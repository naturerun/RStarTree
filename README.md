## 概述:

该项目为C++版本的R*树实现,基于visual studio 2026构建,核心部分只有一个cpp文件,拉取代码后构建解决方案即可.研究代码前务必理解R*树的原理，可以参考github仓库中一篇介绍R树的原始论文和描述R*树相关算法的中文文档,由于该中文文档错误较多，因此还是建议研究R*树的原始论文The R*-tree: an efficient and robust access method for points and rectangles,仓库里已经包含这篇论文

## R*树库的使用方法
为了方便开发和调试,该库只有一个cpp文件,如果你需要把该库用于自己的项目,请修改源代码结构,采用声明和使用分离的模块化构建方式.库的核心为
RStarTree类,模板参数T为界定多维立方体每一维度坐标范围的数值类型(必须为内置类型),构造函数的参数dm为R*树节点中每一个多维立方体的维度,min为每一个R*树节点的最小子树数目,max为一个R*树节点的最大子树数目,z为类型T的数值0,o为类型T的数值1，f的含义见源代码中RStarTree类声明中first_p的注释
insertData函数用于插入一个多维立方体,参数be_inserted表示被插入的多维立方体,每一个pair表示多维立方体每一维度的坐标范围
remove用于删除一个多维立方体,key的含义和be_inserted相同
isRTree用于验证RStarTree类的对象是否代表一个合法的R*树
具体在代码中的使用方法请参考RStarTree.cpp文件中main函数中的代码，这是掌握该库的使用方法的极好的研究方式

## 构建方式
直接将仓库拉取至本地用visual studio 2026打开RStarTree.slnx构建解决方案运行即可
